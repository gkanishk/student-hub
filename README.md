### A portal for students to get updates and apply for scholarship.

## Stacks used:
- React
- React-Bootstrap
- React-Router
- Express
- Firebase 

## Installation:

```
git clone https://github.com/gkanishk/student-hub
cd student-hub
yarn
yarn start

```
Will load at:

`localhost:3000`
